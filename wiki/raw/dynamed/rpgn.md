# Overview of RPGN (DynaMed)

Updated 29 May 2023. 22 pages.

## Overview
Clinical syndrome: rapid GFR decrease (weeks-months) with nephritic features. Histologically = crescentic GN (crescents in >50% of glomeruli). 3 types by IF: type 1 linear (~10%, anti-GBM), type 2 granular (15-20%, immune complex), type 3 pauci-immune (60-80%, most common, ANCA-associated). Crescentic GN prevalence: 3.73% of biopsies in China (2011-2016). 27-38% of pauci-immune type are ANCA-negative.

## Types
Type 1 (linear): anti-GBM disease (85% of type 1); also Alport post-transplant (3-12%). Type 2 (granular): post-infectious (16%), lupus nephritis (8-13%), HSP (3-10%), IgAN (4%), cryoglobulinemia (11%), MPGN. Type 3 (pauci-immune): MPA, GPA, EGPA (collectively = ANCA-associated vasculitis accounting for ~50% of crescentic GN). ANCA-negative pauci-immune: 27-38%.

## Presentation
May be asymptomatic (urinary screening). Common: hematuria (dysmorphic, microscopic), proteinuria, fatigue, fever, weight loss, flu-like symptoms. Extrarenal symptoms by cause: anti-GBM (hemoptysis, severe anemia), lupus (rash, arthritis), HSP (purpura, abdominal pain), GPA (sinusitis, nodules, mononeuritis), MPA (pulmonary infiltrates, skin ulcers).

## Complement Levels
Anti-GBM: normal/normal. MPGN: low/low or normal. Cryoglobulinemia: normal or low/low. Lupus: low/low. IgAN: normal/normal. Post-streptococcal: low/normal or low. DDD: low/normal. ANCA: normal/normal.

## Biopsy
Crescents: proliferative lesions >=10% Bowman capsule circumference. Cellular (>75% cells, potentially reversible). Fibrocellular (25-75%, partially reversible). Fibrous (<25% cells, irreversible). Type 1: crescents + extracapillary proliferation, no intracapillary; linear IgG on IF. Type 2: crescents + intracapillary proliferation + fibrinoid necrosis; granular Ig+complement. Type 3: crescents + fibrinoid necrosis + granuloma (GPA); absent/faint IF.

## Treatment
Type 1: pulse steroids + CYC + PLEX (emergency, start before confirmation). Type 2: treat underlying disease. Type 3: high-dose steroids + CYC or rituximab; rituximab = primary choice for pauci-immune [Clinicians' Practice Point]; PLEX if alveolar hemorrhage or severe renal vasculitis; maintenance rituximab q6 months preferred over azathioprine. KDIGO: start treatment immediately if suspected. For older adults with pauci-immune: lower CYC doses, shorter steroid course, consider rituximab.

## Prognosis
5-year renal survival: type 1 (linear) 17.6%, type 2 (granular) 70.1%, type 3 (pauci-immune) 44.3%. Poor prognostic factors: older age, elevated Cr at presentation, sclerotic ANCA histopathology, >80% crescents (type 2), oligoanuria, dialysis dependence. Linear pattern has worst renal prognosis. Mortality in crescentic GN: 30.7% during median 458-day follow-up. Infection is most common cause of death in immunosuppressed patients.
