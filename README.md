# 腎專考試複習專區

TSN 腎臟專科醫師考試準備資源。核心入口是互動式 Q-bank：[`docs/qbank/`](./docs/qbank/)。

## 內容

| 區塊 | 內容 |
|---|---|
| [`cme/`](./cme) | 4-option A-D 題庫、answer reveal、rationale、distractor analysis |

| [`nephrology-cme-wiki/`](./nephrology-cme-wiki/) | 題目 rationale 的 public-safe 知識索引 |

## Public-Safety Policy

- 不收病人資料。
- 不放 PDF、原始 textbook 內容、付費期刊全文、考古題原文或 NephSAP 題幹原文。
- 題目與解析必須是原創/改寫後的教育內容。
- 非 open-access 或授權不明來源只作 bibliographic citation，不公開原文。

## 使用方式

1. 開啟 [`docs/qbank/`](./docs/qbank/) 練習題目。
2. 直接閱讀 `cme/` Markdown。
3. 用 GitHub 搜尋主題、疾病、藥物或 guideline 名稱。
